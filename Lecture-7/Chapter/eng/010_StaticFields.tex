\begin{frame}{Static Arrays}
  \begin{itemize}
    \item
      The name \enquote{static array} has nothing to do with the keyword
      \lstinline[
        language=Java,
        style={java-eclipse-code},
        basicstyle=\small
      ]|static| from Java / C++
    \item
      Nor is the array allocated before the program starts
    \item
      The {\color{Mittel-Blau}size} of the array is static and can not be
      changed after creation
    \item
      The name \enquote{fixed-size array} would be more appropiate
  \end{itemize}
\end{frame}
